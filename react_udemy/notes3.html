Web Storage

Cookie: Valores que podemos salvar no navegador. (Ex.Ultima página que acessou, ultimo dia etc...) o cookie fica salvo

Sessions: São temporárias, armazena valores enquanto o navegador está aberto (Ex. logout do banco ao fechar o navegador)

Então existem duas formas de salvar no webstorage, o localStorage e o sessionStorage

Pra guardar itens no localStorage usamos o localStorage.setItem("chave", "valor") ou localStorage.chave = "valor"

Pra capturar itens no localStorage usamos o localStorage.getItem("chave) ou localStorage.chave = "valor"

Para excluir itens no localStorage usamos o localStorage.removeItem

Os dados ficam salvos mesmo quando o navegador é fechado.
